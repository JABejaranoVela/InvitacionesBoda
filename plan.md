Dirección visual

  - Estilo: elegante, cálido, editorial; fotos grandes, tipografía serif para títulos.
  - Paleta: crema/beige cálido + grises suaves + verde oliva oscuro para overlay.
  - Tipografía: Serif display para títulos (p. ej. “Playfair Display”), Sans limpia para cuerpo (p. ej. “Inter”).
  - Espaciado: secciones amplias, mucho aire; cards con padding generoso.
  - UI feel: boutique / wedding studio.

  ———

  Páginas y secciones (detalle)

  1. Home

  - Navbar: Logo izquierda, links derecha (Home, Plantillas, Precios, FAQ, Contacto), botón “Ordenar”.
  - Hero:
      - Fondo full-bleed con imagen (la de referencia).
      - Overlay oscuro suave.
      - H1 grande: “Rayakan Momen Berharga…”
      - Subtexto breve y CTA doble (Ver Plantillas / Ordenar).
  - Section: “Design Undangan” (Catálogo destacado)
      - Grid 3–4 cards de plantilla.
      - Cada card: miniaturas, nombre, precio, botón “Preview” y “Order”.
  - Section: Beneficios
      - 3–4 columnas con iconos (Personalizable, Envío rápido, RSVP, Música).
  - Section: Cómo funciona
      - 3 pasos con números grandes.
  - Section: Testimonios
      - Carrusel simple o grid 2.
  - CTA final
      - Bloque ancho con copy “Haz tu invitación hoy”.
  - Footer
      - Logo, links, redes, datos contacto.

  2. Catálogo (Plantillas)

  - Hero corto: título + filtros.
  - Filtros: estilo (minimal, floral, clásico), rango precio.
  - Grid de cards: 3–4 columnas.
  - Paginación.

  3. Detalle de plantilla

  - Galería: mockups + slider.
  - Info: nombre, precio, descripción, incluye (maps, RSVP, música).
  - CTA: “Order” + “Preview”.
  - Sección FAQ mini.

  4. Checkout / Contacto (solo UI)

  - Formulario de datos pareja, fecha, mensaje, email, WhatsApp.
  - Resumen de plantilla seleccionada.

  5. FAQ

  - Acordeones.

  ———

  Sistema de componentes

  - Button: primary (oscuro), secondary (borde), ghost.
  - Card: imagen, badge “Popular”, price tag.
  - Badge: “Nuevo”, “Best Seller”.
  - Section Title: serif + subtexto pequeño.
  - Feature Item: icon + label + copy.
  - Testimonial Card: quote + nombre.

  ———

  Responsive

  - Mobile: navbar con burger, hero centrado, 1-col grid.
  - Tablet: 2-col grid.
  - Desktop: 3–4 col, hero con texto alineado izquierda.

  ———

  Estados UI

  - Hover: cards con sombra suave + elevación.
  - Buttons: hover darken, focus ring visible.
  - Empty state: “No templates found”.
